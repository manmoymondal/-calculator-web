# Web Calculator

A cyberpunk-themed calculator built with HTML, CSS and JavaScript.

## Features
- Addition, Subtraction, Multiplication, Division
- Calculation history (last 5 results)
- Keyboard support
- Backspace and clear buttons
- Division by zero error handling

## How to Use
1. Download calculator.html
2. Open in any browser
3. Use mouse or keyboard

## Keyboard Shortcuts
- 0-9 : Numbers
- +, -, *, / : Operators
- Enter : Calculate
- Backspace : Delete
- Escape : Clear
